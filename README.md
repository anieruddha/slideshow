# slideshow assignment

####Install the dependencies...

```bash
npm install
```

#### Run tests
```bash
npm run test
```

#### Running code in local

```bash
npm run dev
```

#### Build code

```bash
npm run build
```

App will be running on [localhost:5000](http://localhost:5000).
